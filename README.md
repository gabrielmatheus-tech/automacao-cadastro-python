# 🤖 Automação de Cadastro com Python

Projeto de automação desenvolvido em Python utilizando PyAutoGUI e Pandas para realizar login automático em um sistema web e cadastrar produtos automaticamente através de uma planilha CSV.

## 🚀 Funcionalidades

* Abertura automática do navegador
* Login automático
* Leitura de planilha CSV
* Cadastro automático de produtos
* Preenchimento automático de formulários
* Scroll automático da página

---

## 🛠️ Tecnologias utilizadas

* Python
* PyAutoGUI
* Pandas

---

## 📚 Objetivo do projeto

Praticar:

* automação com Python
* manipulação de dados
* leitura de arquivos CSV
* loops
* automação de tarefas repetitivas
